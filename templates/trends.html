Using Machine Learning to Predict Emissions

Climate Change data from World Bank database is used to analyze the factors that can influence greenhouse gas emissions. A hypothesis is formulated to test if CO2 emissions depend on country-specific features available in the dataset and can be predicted from these. Two Machine Learning algorithms: Decision Tree Regressor and Random Forest Regressor models are built to predict CO2 emissions from growth indicators like fossil usage, Gross Domestic Product (GDP) and population.

Decision Tree Regressor Metrics

R-Squared 0.9437
Random Forest Regressor Metrics
Metric Value
Root Mean Squared Error 0.0355

  • Random Forest Regressor fits the data better than Decision Tree Regressor.
  • The weighted average error between the predictions and actuals in this dataset is 0.0355, which is likely a fair value given that 1.36 (kt) is the average actual emissions per capita.
  • In conclusion, Random Forest Regressor allows prediction of CO2 emissions per capita from country specific indicators like electricity access percent, fossil usage, urban population and GDP.
